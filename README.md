# All Result BD

বাংলাদেশের সকল বোর্ডের শিক্ষার্থীদের ফলাফল দেখার জন্য একটি ওয়েব অ্যাপ্লিকেশন।

## পাতা সমূহ
- `index.html` – সাধারণ ব্যবহারকারীদের জন্য রেজাল্ট দেখার পাতা
- `admin.html` – প্রশাসকের জন্য রেজাল্ট আপডেট ও ব্যবস্থাপনার পাতা

## প্রযুক্তি
- HTML5
- CSS3  
- Bootstrap 5
- JavaScript / jQuery

## লাইভ সাইট
https://all-result-bd.netlify.app/
## ভবিষ্যৎ পরিকল্পনা
- মোবাইল অ্যাপ সংস্করণ
- এসএমএসের মাধ্যমে রেজাল্ট পাঠানোর ব্যবস্থা
